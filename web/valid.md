## 2. Критичные ошибки (влияющие на отображение или функциональность)

### 1. Отсутствие атрибута `alt` у изображений (3 ошибки)

Сообщение валидатора:
> An "img" element must have an "alt" attribute.

Проблема:

- У нескольких тегов `<img>` отсутствует обязательный атрибут `alt`.
- Это является нарушением стандартов HTML5.
- Влияет на доступность (screen readers) и SEO (Search Engine Optimization).

Влияние:

- Нарушение доступности
- Потенциальные проблемы с индексацией
- Некорректная работа вспомогательных технологий

Приоритет: **Высокий**

---

## 3. Предупреждения

### Устаревший атрибут `frameborder` у `<iframe>`

Сообщение валидатора:
> The "frameborder" attribute on the "iframe" element is obsolete. Use CSS instead.

Проблема:

- Используется устаревший HTML-атрибут.
- Следует заменить на CSS-свойство `border`.

Влияние:

- Не критично для отображения
- Нарушение современных стандартов HTML

Приоритет: **Средний**

---

## 4. Наиболее частые типы ошибок

### 1. Лишний закрывающий слэш у void-элементов

Сообщение:
> Trailing slash on void elements has no effect and interacts badly with unquoted attribute values.

Пример:

```html
<link rel="alternate" hreflang="en-US" href="https://www.pathofexile.com/" />
```

Проблема:

- В HTML5 закрывающий слэш у void-элементов (`<link>`, `<img>`, `<meta>` и др.) не требуется.
- Это не ошибка синтаксиса, но не соответствует современному стандарту HTML.

Влияние:

- Не влияет на отображение
- Относится к качеству кода

Приоритет: **Низкий**

## 5. Ошибки доступности

Выявлены проблемы доступности:

- Отсутствие `alt` у изображений
- Возможные декоративные изображения без корректного `alt=""`

Сайт частично нарушает принципы доступности (WCAG). Основная проблема — отсутствие текстовых альтернатив для изображений.

Приоритет исправления: **Высокий**
